--- MODO DE USAR ---
1. O individuo deve inserir, primeiramente, o seu nome completo conforme o documento de identidade NO CAMPO ADEQUADO.
2. O individuo deve inserir, consequentement, o seu cargo em acordo com as normas da constituição o qual ele pertence, NOVAMENTE NO CAMPO ADEQUADO.
3. O individuo deve inserir, finalmente, o seu departamento conforme assinalado pela constituição o qual ele pertence. Nesse caso, a SCP Foundation possui somente três departamentos distintos: Marketing, Desenvolviment e Design.
4. Certifique-se que todos os campos foram preenchidos adequadamente pelo usuário, pois ao apertar o botão de gerar crachá, o código não irá reconhecer caso não tenha quaisquer caracter em pelo menos um campo.
5. Veja a mágica acontecer!
